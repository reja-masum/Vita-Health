💾 Backup & Restore

A .json backup file containing all data will be created weekly. Clicking ☁ Upload to Drive will automatically save it to Google Drive via Apps Script.
⚙ Setup: Apps Script → Project Settings → Script Properties → Add: BACKUP_FOLDER_ID = your Drive backup folder ID. If not set, it will save to root Drive.
